# Pi RPC validation fixtures

These JSONL fixtures are sanitized copies of the OpenRouter Pi RPC validation transcripts captured under the local validation temp directory. Machine-local paths were replaced with placeholders such as `<PI_SESSION_DIR>` and `<VALIDATION_WORKDIR>`.

GitHub Copilot is assumed to be event-compatible with the validated OpenRouter Pi RPC event stream and is not a blocker for the v1 Pi harness implementation.
